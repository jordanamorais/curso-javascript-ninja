### Data Types

Basically _number_, _boolean_, _string_, _null_ , _undefined_ and **objects** ( {} )
and **arrays** ( [] ).

> Objects:

**OBS:** Objects are most faster than arrays. So, prefer use objects instead of arrays.

```js
var person = {
  height: 1.87,
  weight: 90, //kg
}

// To access object values:
person.height;
person.weight;
// or
person["altura"];
person["weight"];
```

> Array

**OBS:** Arrays in JS are _objects_.

```js
// Empty array
var numbers = [];

// Array with numbers
var numbers = [1, 2, 3, 4];

numbers[0]; // = 1
numbers[1]; // = 2
```
